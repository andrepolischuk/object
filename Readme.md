
# object

  Object keys / values / length.

## Example

```js
var is = require('is');
is.date(obj);
```

## API

### .keys(obj)

  Return the keys for `obj`.

### .values(obj)

  Return the values for `obj`.

### .length(obj)

  Return the number of keys for `obj`.

### .isEmpty(obj)

  Check if `obj` is empty.
